# .vim
This is my setup for vim. To install, run the following commands:
```
$ cd ~/
$ git clone --recursive https://github.com/joncrawf/.vim.git .vim
$ ln -sf $HOME/.vim/vimrc $HOME/.vimrc
```

## Plugins
